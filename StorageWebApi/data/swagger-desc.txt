## Api проекта для имитации управления складом. Создано в учебных целях.

Для работы требуется подключение к созданной базе данных


### Сущности используемые в API ###
    
| Сущности          | Описание                                                                       |
| ----------------  | -------------------------------------------------------------------------------|
| Area              | Зона, где хранятся товары. В ней находятся SubAreas и Cells                    |
| SubAreas          | Стеллаж или шкаф, где находятся ячейки (Cells) с товарами                      |
| Cells             | Тут хранятся товары. Размеры и параметры ячейки задаются в CellTypes           |
| CellTypes         | Тип ячейки. Тут задаются ее размеры, максимальный вес для храения              |
| Cells             | Тут хранятся товары. Размеры и параметры ячейки задаются в CellTypes           |
|                   |                                                                                |
| Categories        | Категория товара                                                               |
| Products          | Описание товара, его вес, категория, размеры и цена                            |
| Items             | Товар, который храниться в ячейке. Имеет описание (ProductId) и ячейку (CellId)|


### Как работать

1. **Создайте тип ячейки**, где будет храниться ваше добро (CellType)

   ```POST /api/Storage/CellTypes```
   

2. **Затем нужно создать Зону** (Area) используя: 

   ```POST /api/Storage/Area```

   Назовите свою зону в поле name как хотите
   В массиве subAreasToCreate опишите размеры (в ячейках) и типы стеллажей которые хотите создать,
   их может быть несколько.
   
   * Ваша первая зона храения создана, теперь в нее можно класть ваши товары.
     В разделах Area, Cells при помощи ```GET``` методов вы можете просматривать информациию о вашей зоне
   

3. **Добавим товар.** Создайте категорию товара при помощи: 

   ```POST /api/Products/Categories```
   

4. **Создаейте описание товара** при помощи: 

   ```POST /api/products```
   
5. Теперь **добавим товар в ячейку** при помощи: 

   ```POST /api/Storage/Items```
   В теле укажите id описания товара, id ячейки для храения, и дату добавления (если нет, оно создатся на сервере)

Вот и все. Храните, добавляйте и перемещайте товары.
